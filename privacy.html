Privacy Policy for Tetdir

Last updated: November 20, 2025

Introduction

Tetdir provides tools to connect creators' social profiles, stores, and payment methods. This page explains what data we access and how we use it.

Data we access

We use TikTok Login to retrieve the following basic user information after you authorize our app:

  • Display name / username
  • Profile avatar
  • Unique user id (for login and account link)

No sensitive personal data (like phone numbers) is collected by default.

How we use the data

We use the data to authenticate users, show profile info inside Tetdir, and link connected services. We do not sell user data. Users may disconnect TikTok from their Tetdir account at any time.

Third-party services

We may use third-party payment providers and hosting (e.g., GitHub Pages for this policy). These providers' privacy terms apply separately.

Contact

If you have questions, contact: hello@tetdir.com
